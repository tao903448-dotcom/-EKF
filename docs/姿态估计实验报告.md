# 四旋翼姿态估计 EKF —— 实验报告

> 第十五届中国软件杯 · A8 四旋翼无人机位姿控制系统设计优化
> 版本：2026-06-21 ｜ 复现：`make run-attitude`

---

## 1. 问题与定位

A8 题目的核心是**四旋翼无人机位姿估计**。本项目据此实现一套基于 IMU
（陀螺 + 加速度计 + 磁力计）的姿态估计 EKF，并把项目的算法创新——
**面向模型失配的自适应 / 鲁棒滤波**——直接作用于该真实问题，定量回答：
*在振动野值、机动失配等真实干扰下，自适应 / 鲁棒更新能带来多少收益？*

## 2. 滤波器建模

| 项 | 设计 |
|---|---|
| 状态 x (7) | 姿态四元数 q（body→world, Hamilton）+ 陀螺零偏 b (rad/s) |
| 控制 u (3) | 陀螺测量 ω_m (rad/s) |
| 预测 | q ← q ⊗ Δq(​(ω_m−b)·dt)，b 随机游走；每步重归一化(框架 state_normalize 钩子) |
| 观测 z (6) | 加速度计"重力方向"单位向量 + 磁力计"参考方向"单位向量（均 body 系） |
| 观测模型 | h(x) = [R(q)ᵀ·ĝ_w ; R(q)ᵀ·m̂_w] |
| 雅可比 | 中心差分**数值雅可比**（correct-by-construction，避免手推易错） |

设计取舍：
- **四元数**避免欧拉角万向锁，适合大机动；采用"加性 EKF + 重归一化"以
  复用通用框架的全部更新方法（严格乘性 MEKF 误差状态形式列为后续增强）。
- **方向型观测**：加速度计在机动（比力≠重力）时，其"重力方向"天然偏离真值，
  恰好构成**模型失配**——这正是自适应 / 鲁棒更新的考点。

## 3. 实验设置

- 仿真：可复现 IMU 仿真器（`include/imu_sim.h`，确定性伪随机），200 Hz，15 s。
- 蒙特卡洛：每格 20 个随机种子取平均。
- 噪声：陀螺白噪声 ~0.57°/s + 零偏随机游走；加速度/磁方向噪声 σ≈0.02。
- 标定：观测噪声 R = 4×10⁻⁴（与真实方向噪声方差一致）。
- 三场景：
  1. **CLEAN**：温和机动、无野值；
  2. **OUTLIER**：注入 12% 加速度脉冲野值（电机振动）；
  3. **MANEUVER**：大机动 + 6 m/s² 线加速度（比力严重偏离重力）。
- 指标：稳态**姿态 RMSE（度）**、陀螺**零偏 RMSE**、平均 **NIS**（新息一致性）、
  **NIS 越界比例**（超 95% χ² 上界）。
  *注：单位方向观测受单位范数约束，每个 3 维观测有效自由度≈2，两个观测合计≈4，
  故一致滤波的平均 NIS 应≈4 而非 6。*

## 4. 结果

| 场景 | 方法 | 姿态RMSE(°) | 零偏RMSE | 平均NIS | NIS越界% |
|---|---|---|---|---|---|
| **CLEAN** | 标准EKF | 0.421 | 0.0047 | 3.80 | 4.1 |
| | Joseph | 0.421 | 0.0047 | 3.80 | 4.1 |
| | Student-t | 0.429 | 0.0047 | 4.21 | 0.0 |
| | 自适应 | **0.410** | 0.0047 | 3.39 | 0.0 |
| **OUTLIER** | 标准EKF | 1.746 | 0.0066 | 72.97 | 28.8 |
| | Joseph | 1.746 | 0.0066 | 72.97 | 28.8 |
| | Student-t | 0.439 (**↓75%**) | 0.0055 | 5.09 | 11.7 |
| | 自适应 | **0.420 (↓76%)** | 0.0055 | 3.99 | 2.6 |
| **MANEUVER** | 标准EKF | 25.68 | 0.078 | 199.6 | 99.1 |
| | Joseph | 25.68 | 0.078 | 199.6 | 99.1 |
| | Student-t | 18.60 (↓28%) | 0.079 | 13.6 | 99.1 |
| | 自适应 | **15.54 (↓39%)** | 0.077 | 8.83 | 28.8 |

## 5. 分析

1. **CLEAN**：四法接近，姿态 RMSE≈0.42°，平均 NIS≈3.8≈有效自由度，**滤波器一致**；
   `标准 ≡ Joseph`（对最优增益数学等价）佐证实现正确。
2. **OUTLIER**：标准 EKF 被振动野值污染，RMSE 升至 1.75°、NIS 飙到 73（严重过自信、
   不一致）；Student-t / 自适应通过对大马氏距离观测降权，**RMSE 降低 75–76%**，
   并把 NIS 拉回≈4（**恢复一致性**）。这说明收益不仅在精度，更在估计的可信度。
3. **MANEUVER**：线加速度使加速度计"重力方向"持续失配，标准 EKF 误差达 25.7°；
   自适应据 NIS 在机动期自动放大 R、转而依靠陀螺，**误差降低 39%**。
   但加速度计在持续线加速度下本质上失去重力信息，仅靠陀螺(漂移)+磁力计(偏航)
   无法完全补偿，故残留误差较大——这是**诚实的物理上限**，进一步改善需引入
   运动模型 / GPS 等外部信息（后续工作）。

**结论：不存在"永远最优"的更新方法；自适应 / 鲁棒更新在其对症场景
（测量野值、观测失配）下能显著且可验证地提升四旋翼姿态估计的精度与一致性。**

## 6. 复现

```bash
make run-attitude                 # 打印上表
./build/attitude_demo traj.csv    # 导出 MANEUVER 下"标准 vs 自适应"轨迹(roll/pitch/yaw)
python3 tools/plot_attitude.py traj.csv   # 出对比图(需 matplotlib)
make test                         # 含姿态/四元数回归测试(tests/test_attitude.c)
```

## 7. 工程要点

- 姿态模型作为通用 EKF 框架的一个配置（`src/attitude.c`），四种更新方法零成本复用；
- 为带约束状态新增 `state_normalize` 钩子（四元数重归一化），框架更通用；
- 数值雅可比使复杂非线性模型免于手推，降低出错面；
- 提供 CI 配置 `ci/ci.yml`（GitHub Actions：x86 构建+测试、ASan/UBSan、
  ARM(NEON) 交叉编译），复制到 `.github/workflows/` 即可启用。

# 面向模型失配的自适应EKF系统 - 技术文档

## 第十五届中国软件杯 - A8四旋翼无人机位姿控制系统设计优化

---

## 1. 项目概述

### 1.1 项目背景
本项目针对四旋翼无人机位姿控制中的状态估计问题，设计并实现了一套面向模型失配的自适应扩展卡尔曼滤波器（EKF）系统。通过对比标准EKF、Student-t鲁棒更新和自适应EKF的性能，展示不同滤波策略在无人机姿态估计中的效果差异。

### 1.2 核心特性
- **多策略EKF更新**：标准/Student-t/自适应/Joseph四种方法
- **Student-t鲁棒滤波**：抑制传感器野值干扰
- **自适应噪声估计**：基于新息序列动态调整噪声协方差
- **零动态分配矩阵运算库**：适用于嵌入式实时系统

### 1.3 应用场景
- 四旋翼无人机姿态估计
- 惯性导航系统
- 目标跟踪与定位
- 机器人状态估计

## 4. EKF框架

### 4.1 算法原理

#### 标准EKF方程

**预测步骤：**
```
x̂(k|k-1) = f(x̂(k-1|k-1), u(k))
P(k|k-1) = F(k) · P(k-1|k-1) · F(k)ᵀ + Q(k)
```

**更新步骤：**
```
K(k) = P(k|k-1) · H(k)ᵀ · [H(k) · P(k|k-1) · H(k)ᵀ + R(k)]⁻¹
x̂(k|k) = x̂(k|k-1) + K(k) · [z(k) - h(x̂(k|k-1))]
P(k|k) = [I - K(k) · H(k)] · P(k|k-1)
```

### 4.2 更新方法

#### 1. 标准EKF更新
- 最简单的更新方法
- 计算复杂度：O(n³)
- 适用于小规模系统

#### 2. Joseph形式更新
- 数值稳定的协方差更新
- 公式：P = (I-KH)P(I-KH)ᵀ + KRKᵀ
- 保证协方差矩阵对称正定

#### 3. Student-t鲁棒更新
- 使用Student-t分布替代高斯分布
- 基于马氏距离检测野值，自动降低野值权重
- 自由度参数ν控制鲁棒性
- 抗传感器野值干扰能力强

#### 4. 自适应更新
- 基于新息序列的协方差匹配算法
- 动态调整测量噪声协方差R
- 自适应因子：R_adaptive = R × (1 + α × max(0, innov²-1))
- 应对模型失配和参数摄动问题

### 4.3 核心API

### 8.3 使用示例

```c
#include "matrix.h"
#include "ekf.h"

int main() {
    // 初始化EKF配置
    EKF_Config config;
    ekf_config_init(&config, 2, 1);

    // 初始化状态
    Matrix x0, P0;
    matrix_init(&x0, 2, 1);
    matrix_init(&P0, 2, 2);

    EKF_State ekf;
    ekf_state_init(&ekf, &config, &x0, &P0);

    // 滤波循环
    for (int i = 0; i < N; i++) {
        // 预测
        ekf_predict(&ekf, &config, &u, dt);

        // 更新
        ekf_update(&ekf, &config, &z);

        // 获取估计
        Matrix x_est;
        ekf_get_state(&ekf, &x_est);
    }

    return 0;
}
```

## 9. 项目创新点

### 9.1 四种EKF算法对比验证（诚实评测）
- 在相同传感器数据下对比标准 / Joseph / Student-t / 自适应。
- **关键结论：不存在"永远最优"的方法，需按噪声特性选型。**
  - 过程模型失配（无野值）场景：标准 EKF 最优，R 膨胀类方法反而更差。
  - 测量野值场景：鲁棒/自适应方法显著占优（见 9.2 / 9.3）。
- `Standard ≡ Joseph`（对最优增益数学等价），可作为实现正确性的自检。

### 9.2 Student-t鲁棒滤波
- 基于马氏距离 `d² = y'S⁻¹y` 检测野值；`R_eff = R·(ν+d²/δ²)/(ν+n)`，
  野值 → R 膨胀 → 增益收缩，自动降权。
- **在"匀速模型 + 15% 脉冲野值"场景，RMSE 相对标准 EKF 降低约 89%**
  （`make run-demo` 可复现）。

### 9.3 自适应噪声估计
- 基于归一化新息平方 `NIS = y'S⁻¹y`（多维有效）动态调整 R：
  `factor = 1 + α·max(0, NIS/n − 1)`，并夹在 `[0.1, window]`。
- 同一野值场景下 **RMSE 降低约 87%**。

> 说明：早期文档曾笼统宣称"Student-t 降低 43%、自适应降低 65%"。经核查，
> 该结论受"标准 EKF 协方差坍缩 bug"与"演示算法标签错位"双重影响而失真；
> 重构版以诚实的双场景基准重新给出上述数据。详见
> `docs/深度剖析与优化报告.md`。

### 9.4 零动态分配嵌入式设计
- 所有内存在编译时确定，避免 malloc/free，适合飞控实时运行。
- 矩阵运算 alias-safe，且 NEON 与标量同处一份 `matrix.c`，ARM 构建可用。
